# nzv tools

Advanced terminal multitool.

## Install

```
git clone https://github.com/c7s89r/nzv-tools.git
cd nzv-tools
pip install -r requirements.txt
python main.py
```

## Features

- Discord operations (webhooks, tokens, nuke bot)
- OSINT & network tools (port scanner, whois, dns, phone/email lookup)
- Builder suite (stealer, clipper, keylogger, RAT)
- Roblox utilities
- Faker & simulation modules
- Crypto tools, obfuscator, metadata scanner
- Proxy tools
